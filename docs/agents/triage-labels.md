# Triage labels

这些技能内部使用五个标准 triage 角色。本文件把这些角色映射到本仓库 issue tracker 中实际使用的 label 字符串。

| 技能中的标准角色 | 本仓库使用的 label | 含义 |
| --- | --- | --- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，可交给 AFK agent 处理 |
| `ready-for-human` | `ready-for-human` | 需要人类实现 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色时，使用上表右侧对应的 label 字符串。

如果以后仓库改用不同的 GitHub label 名称，直接修改本文件右侧列即可。
