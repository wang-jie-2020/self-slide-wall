# 观众提问墙

本上下文描述一个用于演讲、分享等展示型活动的实时观众互动产品。产品建模的是现场互动活动，而不是演讲本身。

## 语言

**观众提问墙**：
一个实时网页体验，观众可以在演讲过程中提交问题，并通过点赞表达哪些问题最值得被关注。
_避免_：会议工具、协作工具

**近实时更新**：
活动更新通常会在 1-2 秒内出现在各个视图中，并且不需要用户手动刷新页面。
_避免_：即时更新、离线同步

**目标活动规模**：
第一版对单个活动的规模目标：约 200 个活跃观众会话正在查看页面、提交问题、点赞问题和参与投票。
_避免_：连接数、流量目标

**观众问题**：
由观众会话在活动中提交的问题。观众提交后不能编辑或删除。
_避免_：评论、消息

**问题字数限制**：
某个活动中观众问题文本的最大长度，默认 240 字符。
_避免_：帖子限制、消息限制

**演讲**：
现实世界中的展示型场景，通常由一位或少数几位演讲者面对观众进行分享。演讲是活动的背景，不是产品直接管理的对象。
_避免_：会议、场次

**活动**：
一次现场互动实例，由一个主持账号拥有，观众通过访问码或链接加入。
_避免_：房间、频道

**草稿活动**：
正在准备中、尚未接受观众互动的活动。
_避免_：已排期活动、未激活活动

**进行中活动**：
正在接受观众问题、问题点赞和投票的活动。
_避免_：活跃活动、开放活动

**已结束活动**：
不再接受观众互动，但仍可以只读访问的活动。
_避免_：归档活动、关闭活动

**已删除活动**：
从主持控制台默认列表中隐藏，且不再能通过观众视图或展示视图访问的活动；其数据可以继续保留。
_避免_：永久删除活动、归档活动

**访问码**：
系统生成的短码，观众可用它加入活动。已结束活动仍可通过访问码只读访问；已删除活动不可再访问。
_避免_：房间码、会议码

**观众**：
参加演讲的人，可以提交问题并为问题或投票选项表达选择。
_避免_：参与者、参会者

**观众会话**：
观众在单个活动中的临时身份，通常在其设备打开提问墙时创建，并在同一浏览器中保留到活动结束。
_避免_：用户、账号、个人资料

**显示名**：
观众会话加入活动时可以提供的可选名称。允许匿名进入，显示名不是已验证身份。
_避免_：用户名、真实姓名

**主持账号**：
产品内唯一操作身份。主持账号可以创建活动并使用主持控制台管理活动、问题和投票；第一版可以使用少量示例主持账号，而不提供完整账号系统。
_避免_：演讲者、控场人、product owner

**主持控制台**：
主持账号用于管理问题、发起投票以及控制公开展示内容的活动控制界面。
_避免_：管理后台、仪表盘

**观众视图**：
面向观众的活动界面，观众可在这里提交问题、点赞问题并参与投票。
_避免_：参与者页面、参会者页面

**展示视图**：
由主持账号打开并投屏的现场界面，观众消费投屏内容。展示视图组合展示加入入口、问题墙、投票结果，并支持主持账号直接进行置顶/取消置顶、标记回答、隐藏问题和关闭投票等控场操作。观众不应知道展示视图地址；展示视图与主持控制台是不同界面，职责边界明确。
_避免_：主持控制台、屏幕共享

**隐藏问题**：
被主持账号从可见问题墙中移除的问题。隐藏问题是第一版中低优先级的安全控场能力。
_避免_：删除问题、驳回问题

**已回答问题**：
活动中被标记为已经处理过的观众问题。
_避免_：已解决问题、已完成问题

**置顶问题**：
由主持控制台选中、显示在正常问题排序之前的观众问题。
_避免_：精选问题、高亮问题

**问题排序**：
可见且未回答问题的排序方式，主要按问题点赞数排序，点赞数相同时按提交时间排序。
_避免_：分数、信息流顺序

**问题点赞**：
观众会话表达“我也关心这个问题，并希望它被关注”的当前信号。点赞可以取消，但取消不是第一版优先能力。
_避免_：反对票、评分、投票选择

**投票**：
由主持账号发起的结构化问题，观众从预设选项中选择。
_避免_：问题点赞、问卷

**进行中投票**：
正在接受投票选择的投票。一个活动中可以有多个进行中投票，并且产品不设活动投票数量上限。
_避免_：当前投票、实时投票

**已关闭投票**：
不再接受投票选择，但仍可以展示结果的投票。关闭投票不要求已有投票选择（允许 0 票关闭）。
_避免_：已结束投票、归档投票

**投票选项**：
观众在投票中可以选择的预设答案之一。
_避免_：答案、选择

**投票选择**：
观众会话对某个投票选项的当前选择，在投票关闭前可以修改。
_避免_：问题投票、点赞

**单选投票**：
每个观众会话只能选择一个投票选项的投票。
_避免_：多选投票、评分投票

**投票结果**：
投票选择的聚合分布。观众视图不显示聚合结果；展示视图显示百分比结果；主持控制台显示百分比和原始票数。
_避免_：观众结果、私有结果

**活动导出**：
用于活动后复盘的 CSV 导出，包含观众问题、问题点赞、投票、投票选项和投票结果。
_避免_：报告、分析

**投票排序**：
进行中投票在观众视图和展示视图中的展示顺序，由主持控制台控制；默认新创建的投票排在前面。
_避免_：投票集合、展示限制

**观众乐观反馈**：
观众提交问题、点赞或投票后，观众视图立即给出的反馈；活动状态仍以服务端确认为准。
_避免_：离线模式、客户端权威状态
